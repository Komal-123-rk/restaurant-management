# Restaurant Management System

## Description
This project is a Restaurant Management System developed for academic purposes.
It helps manage food orders, billing, and customer details efficiently.

## Features
- Add / View food items
- Take customer orders
- Generate bill
- Simple and user-friendly interface

## Technologies Used
- Database: (MySQL / SQLite etc.)

## How to Run
1. Download or clone the repository
2. Open the project in your IDE
3. Run the main file
